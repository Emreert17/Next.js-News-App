## ðŸ“¦ Installation

1. Clone the repo:

```bash
git clone https://github.com/Emreert17/Next.js-News-App.git
```

2. Navigate to the project folder (replace news-site with your local folder name if different):
```bash
cd news-site
```

4. Install dependencies:
 ```bash
npm install
```

5. Create a .env.local file in the root directory and add your News API key:
 ```bash
NEXT_PUBLIC_NEWS_API_KEY=your_api_key_here
```

6. Start the development server:
 ```bash
npm run dev
```
